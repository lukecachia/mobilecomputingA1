IDentify Me

Project Goal

The goal of this application is to enable users to have quick access to important information in case of emergency, especially in cases were the user is unconcious.

Main Functionalities:

  • Create an Account
  • Create a specific QR Code containing personal details such as Name, Surname, Blood Type and Emergency contact number
  • A list of First-aid procedures to be used in case of an emergency
  • A list of emeregency contact numbers such as ambulence and Police which opens the dailer on the smartphone

Critique


Road ID

About Road ID

Road ID is a company which sells safety braclets which contain information about the person in case of an emeregency.

This company also has an app which displays the personal details on the lock screen of a smartphone. Further to this, the app has the functionality of tracking the location and calls emergency services or relatives in case the phone is stationery for a long time.

However, the problem with such app is that this approach has a single point of failure. If the user runs out of battery or forgets to wear the braclet the whole purpose is defeated.

Strava Beacon

About Strava Beacon

Strava is one of the most popular activity tracking app and its used by many athletes both ameature and professional.

The Strava Beacon is a feature which generates a unique URL to be shared by the user with relatives. Through the smartphone GPS, the location of the user is updated periodically. Therefore, the safety contacts can have the location of their relative in case of an emergency.

The issue with this application is that is also depends on a smartphone. The user can leave the phone behind or end up a dead battery. This app will can have issues where there is low signal or not signal at all.
